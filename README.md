EatFit

Descripción:
El proyecto consiste en el desarrollo de una aplicación para gestionar hábitos saludables, que integra el control de la alimentación, el seguimiento del ejercicio físico y el registro de la actividad diaria. Permite planificar comidas, gestionar rutinas de entrenamiento y monitorizar indicadores como pasos, calorías y horas de sueño.

Estructura:
Dentro de app se encontrara:

- Componente que es la carpeta raiz de todos los componentes, los ya creados son el componente de comida, entreno, inicio, perfil, navbar, footer y el de autenticacion, que dentro de este estara los componentes de login y signin

- Dentro de la carpeta services estaran todos los servicios de la web

- Dentro de la carpeta validations estaran todas las validaciones de formularios de la web

Dentro de src se encontrara:

- Las carpetas assest>img que han sido creadas por mi para almacenar todos los recursos graficos que utilizare en el proyecto

Uso del SCSS

En styles.scss: Solo he usado aqui el scss para poner estilos de colores y tipografias que usare en todo el proyecto en clases para facilitarme su uso 

En app.component.scss: Solo he puesto el estilo de texto para una animacion y una animacion de carga
